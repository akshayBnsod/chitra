## How to build ?
Prerequisites 
Make sure you have git, cmake and make installed on your system.
* git
* cmake
* make

```
git clone https://github.com/akshnova/chitra
```
```
mkdir chitra/build
```
```
cd chitra/build
```
```
cmake ..
```
```
make
```
